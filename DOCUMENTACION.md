# Documentación del Proyecto: Medical and Clinical Center

## Descripción General
Este sistema es una plataforma web profesional diseñada para la gestión de citas médicas, con una interfaz moderna basada en un diseño de "Medical and Clinical Center". El sistema permite a los pacientes registrarse, explorar especialidades médicas y agendar citas con cálculos automáticos de costos.

## Tecnologías Utilizadas
- **Backend:** PHP 8.x con PDO.
- **Base de Datos:** MySQL.
- **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (Intersection Observer para animaciones).
- **Frameworks:** Bootstrap 5.3 e Icons.

## Instrucciones de Instalación
1. Copiar la carpeta `citas_medicas` en `htdocs`.
2. Importar la base de datos `citas_medicas` en phpMyAdmin.
3. Importar `import.sql`.
4. Acceder a `http://localhost/clinica/index.php`.
